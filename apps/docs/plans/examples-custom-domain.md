# Example: Custom Domain Setup

## Overview

Complete example showing how to set up a custom domain with SSL certificate.

## What It Demonstrates

- Route53 hosted zone lookup
- Cross-region ACM certificate creation
- CloudFront domain configuration
- DNS record creation (A and AAAA)

## Code Structure

- Complete stack example
- All imports needed
- Step-by-step comments
- Output values

## Key Features

- Single domain setup
- Multiple domain aliases (www and non-www)
- Certificate validation
- DNS automation
