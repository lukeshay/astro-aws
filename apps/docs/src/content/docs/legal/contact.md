# Contact Us

We'd love to hear from you! The best way to get in touch with the Astro AWS team is through our GitHub repository.

## GitHub Discussions

For questions, suggestions, feedback, or general discussions about Astro AWS, please visit our [GitHub Discussions](https://github.com/lukeshay/astro-aws/discussions) page.

### Types of Discussions

- **Q&A**: Ask questions about using Astro AWS
- **Ideas**: Share your ideas for new features or improvements
- **Show and Tell**: Share your projects built with Astro AWS
- **General**: Any other topics related to Astro AWS

## GitHub Issues

For bug reports or feature requests, please [open an issue](https://github.com/lukeshay/astro-aws/issues) on our GitHub repository. When reporting bugs, please include:

- A clear description of the problem
- Steps to reproduce the issue
- Your environment details (Node.js version, AWS region, etc.)
- Any relevant error messages or logs

## GitHub Repository

Visit our [GitHub repository](https://github.com/lukeshay/astro-aws) to:

- View the source code
- Contribute to the project
- Star the repository if you find it useful
- Fork the project for your own modifications

## Response Time

We monitor GitHub discussions and issues regularly. While we strive to respond as quickly as possible, please allow 24-48 hours for a response.

## Stay Connected

- **GitHub**: [https://github.com/lukeshay/astro-aws](https://github.com/lukeshay/astro-aws)
- **Documentation**: [https://www.astro-aws.org](https://www.astro-aws.org)

Thank you for your interest in Astro AWS!
